Strings

Strings default to unicode in Python3 - orginally default to ASCII in Python2. ASCII can only represent 8 bits (1 byte).

How do you create an ASCII string in Python3: 

In [1]: my = u'unicode'
In [2]: type(my)
Out[2]: str
In [3]: string = b'ascii'
In [4]: type(string)
Out[4]: bytes

Writing code that is compatible for the both? 

from __future__ import print_function, unicode_literals

'unicode_literals' will determine Whenever you have a literal string in Python2, it will make it a unicode. You will probably need to do this when outputting a file or writing to a network. You will need to convert them to a bytestream, or ultimately bits; which streams to a file, or the network.
